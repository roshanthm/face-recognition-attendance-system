# Face Recognition Attendance System

## Overview
This project is a **Face Recognition–Based Attendance System** built to work fully offline.  
It allows students/employees to register their face once and automatically marks attendance using a webcam.

The system is designed for **college labs, offices, and small institutions** where privacy and offline reliability are important.

---

## Features
- Face registration using webcam
- Real-time face recognition
- Automatic attendance marking
- Admin dashboard to view attendance
- Fully offline (no cloud, no API keys)
- No storage of unknown faces (privacy-focused)

---

## Tech Stack

### Backend
- Python
- Flask
- OpenCV
- face_recognition (dlib-based)

### Frontend
- HTML
- CSS
- JavaScript

### Database
- SQL (MySQL / SQLite)

---

## How It Works
1. Admin registers a user’s face.
2. Face encodings are stored securely.
3. Webcam scans faces in real time.
4. Recognized faces are matched.
5. Attendance is automatically recorded.

---

## Use Cases
- College attendance
